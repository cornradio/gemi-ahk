# gemi-ahk

这是一个专为“AI 时代”打造的 AHK 脚本管理与循环工具。

### 核心理念
**你只需要会说话，就能实现自动化。**

现在的 AI（如 Gemini）已经可以非常精准地根据你的描述编写 AHK 代码。但 AI 写出的代码通常只是“单次操作”的片段。这个工具存在的意义，就是作为一个**载体**，让你把 AI 生成的代码“扔进来”，交由它来负责最麻烦的**逻辑循环、次数控制、启动/暂停管理**。

### 极速工作流
1. **获取坐标**：鼠标指向目标，按 **F4** 复制屏幕坐标。
2. **描述需求**：把你的需求（点击哪里、等待多久、移动到哪）写成白话发给 Gemini。
   - *示例：*"帮我写个 AHK 2.0 片段：移动到 500,500 点一下，然后移动到 800,800，中间停 1 秒。"
3. **放入脚本**：点击 **Open Folder**，在打开的文件夹里新建个 `.ahk` 文件，把 AI 生成的代码粘贴进去并保存。
   - **注意：** 脚本里**不需要**写 Loop，只需要写那几行基础动作。
4. **一键循环**：在工具里设置循环次数（0 为无限），选中它按 **F1** 开始。

### 为什么用这个工具？
- **告别编写 Loop**：AI 往往不清楚你要循环多少次或如何优雅地停止，这个工具帮你把循环逻辑和业务动作彻底分离。
- **全局掌控**：预设好了全局快捷键，随时开始 (**F1**)、随时暂停 (**F2**)、随时止损 (**F3**)。
- **可视化计数**：你可以清晰看到脚本执行了多少次，不用猜它跑没跑完。
