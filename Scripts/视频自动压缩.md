# gemi-ahk 任务指令模板 (v2.0 稳健版)
## 1. 背景声明
环境：AutoHotkey 2.0 (v2.0.x Standard)。
要求：
- 仅输出业务代码片段，严禁函数体、全局 Loop 或热键。
- 坐标模式：必须使用 `CoordMode "Mouse", "Screen"`。
- 路径兼容：严禁使用 `A_Videos` 等非标准变量。获取视频/文档路径请统一使用 `EnvGet("USERPROFILE") "\Videos"` 这种环境变量方式。
- 语法简洁：尽量减少中间变量，使用最直接的表达式。
- 容错处理：执行文件操作前先静默判断目录是否存在，不要有 MsgBox 弹窗。
## 2. 操作描述（改成你自己的）
- 获取 库\视频 下的最新视频
- 把这个视频复制到 D:\素材 文件夹下
- 执行 "C:\lightspeed\5\code\ffmpeg\ffmpegcovertD.bat" 文件，对 D:\素材 文件夹下的视频进行处理
- 打开 D:\素材\压缩 文件夹 给用户查看最新的结果
